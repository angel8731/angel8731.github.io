# 自助洗车微信小程序（示例版）

本目录提供一个可直接导入微信开发者工具的基础项目骨架，覆盖无人洗车核心流程：

- 用户登录与资料展示（车牌、手机号、会员、余额、积分）
- 地图找洗车点（设备状态：空闲/使用中/故障）
- 扫码启动洗车（选择模式 + 支付/扣会员）
- 团购核销（输入券码示例）
- 会员机制（月卡、次卡、VIP）
- 充值活动、优惠券、订单记录
- 商家后台示例页面（设备、订单、会员、营销）

## 页面结构

- `pages/home`：首页与快捷入口
- `pages/map`：附近网点列表与导航
- `pages/scan`：扫码启动与团购核销
- `pages/member`：会员购买与充值活动
- `pages/orders`：订单/洗车记录
- `pages/profile`：用户中心（余额、积分、优惠券）
- `pages/admin`：商家后台示例

## 对接建议

1. 将 `services/api.js` 的 mock 数据替换为真实后端请求。
2. 设备控制建议通过后端统一下发 MQTT/HTTP 指令，避免小程序直连设备。
3. 支付使用微信支付（JSAPI）+ 服务端签名；支付宝可放在独立 H5/APP 渠道。
4. 团购核销可扩展支持：美团、抖音、自营券码中心。

## 后端推荐模块

- 用户中心：登录、实名、车辆、积分、钱包
- 订单中心：支付单、洗车单、退款单
- 会员中心：月卡、次卡、等级权益
- 设备中心：设备在线状态、远程启停、故障上报
- 营销中心：充值活动、优惠券、团购活动
